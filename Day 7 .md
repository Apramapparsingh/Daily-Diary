# Day 7 - Metasploitable Introduction and Basic Attacks
**Date:** 09 July 2026

## Objective
The objective of today's session was to understand the purpose of **Metasploitable**, learn how to set it up in VMware, identify its IP address, and perform basic penetration testing from Kali Linux in a controlled environment.

## Topics Covered
- Introduction to Metasploitable
- Purpose of vulnerable virtual machines
- Installing and running Metasploitable in VMware
- Finding the IP address of the target machine
- Connecting Kali Linux with the target machine
- Basic reconnaissance and attack process
- Ethical hacking and legal considerations

## Practical Session

Today's practical session focused on **Metasploitable**, a deliberately vulnerable Linux machine designed for cybersecurity training. The trainer explained that it is widely used by ethical hackers and security professionals to safely practice penetration testing techniques without affecting real-world systems.

We downloaded the Metasploitable virtual machine and successfully ran it using VMware. After booting the system, we learned how to identify its IP address, which is required for communication between Kali Linux and the target machine.

Once the IP address was obtained, we switched to Kali Linux and used it as the target for basic penetration testing exercises. The trainer demonstrated the initial steps followed by penetration testers, including identifying the target system, verifying network connectivity, and performing basic reconnaissance before attempting any attack.

The session also included demonstrations of basic attacks on the vulnerable machine. The trainer emphasized that these techniques are strictly for educational purposes and should only be performed on systems where proper authorization has been granted.

## Key Learnings
- Understood the purpose of Metasploitable in cybersecurity training.
- Learned how to deploy Metasploitable in VMware.
- Learned to find the IP address of a virtual machine.
- Understood how Kali Linux communicates with a target machine.
- Gained knowledge of the basic penetration testing workflow.
- Learned the importance of ethical and authorized security testing.

## Conclusion

Today's session provided a practical introduction to penetration testing using Metasploitable and Kali Linux. The hands-on exercises helped me understand how vulnerable systems are used for cybersecurity training and how penetration testers begin the process of identifying and assessing security weaknesses in a safe and controlled environment.
