# Day 8 - Introduction to Metasploitable and Basic Attack Demonstration

**Date:** 10 July 2026

## Objective
The objective of today's session was to understand the purpose of Metasploitable, learn why it is used in cybersecurity training, install it in VMware, and observe how vulnerable machines can be used for ethical hacking practice.

## Topics Covered
- Introduction to Metasploitable
- What is a vulnerable machine?
- Purpose of Metasploitable in cybersecurity
- Downloading and setting up Metasploitable in VMware
- Basic attack demonstration on a vulnerable machine
- Importance of practicing attacks in a controlled lab environment

## Practical Activities
- Downloaded the Metasploitable virtual machine.
- Imported and configured the virtual machine in VMware Workstation.
- Successfully booted the Metasploitable operating system.
- Learned that Metasploitable is an intentionally vulnerable machine designed for cybersecurity training and penetration testing.
- Understood that it is commonly used as a target machine to practice attacks safely without affecting real systems.
- Observed a basic attack demonstration performed by the instructor on the Metasploitable machine to understand how attackers identify and exploit vulnerabilities.
- Learned that all security testing should be performed only in authorized and controlled environments.

## Learning Outcome
Today's session helped me understand the concept of vulnerable machines and their importance in ethical hacking. I learned how to install and run Metasploitable in VMware and understood that it is intentionally designed with security weaknesses so students can safely practice penetration testing techniques. I also observed a basic attack demonstration, which gave me an introduction to how vulnerabilities can be identified and exploited in a controlled lab environment.

## Conclusion
Today was an important milestone in my cybersecurity training. I successfully installed and ran the Metasploitable virtual machine in VMware and learned why it is widely used for ethical hacking practice. The demonstration of a basic attack helped me understand how cybersecurity professionals test vulnerable systems to improve security. This practical session provided a strong foundation for future penetration testing and vulnerability assessment exercises.
